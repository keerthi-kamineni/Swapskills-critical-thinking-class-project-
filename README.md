Skill Swap – Trade Skills, Not Money

Overview
Skill Swap is a startup idea that enables people to learn new skills without spending money by exchanging knowledge with others.
Instead of paying for expensive courses users can:
Teach what they know Learn what they want

Problem Statement 
 High cost of learning new skills
 Limited access to quality resources
 No proper platform for skill exchange

Solution
Skill Swap connects users through a peer-to-peer skill exchange system

Example:
 You teach Guitar 
 You learn Coding 
No money involved ,just mutual learning
